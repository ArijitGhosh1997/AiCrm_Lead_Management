# QA Test Cases & Bug Reports – AI CRM & Lead Management System

This repository showcases QA testing experience on an AI-driven sales automation platform involving CRM integrations, chatbot interactions, and multi-channel communication workflows.

---

## Scope of Testing

- Lead Lifecycle Management  
- AI Chatbot Interaction & Response Validation  
- Outbound Communication (Email, SMS)  
- API Testing (AI + Backend)  
- UI Validation  
- Database Verification  
- CRM Integration Testing  

---

## Lead Lifecycle Coverage

The system operates across multiple lead stages:

- Not Contacted Yet  
- Initial Engagement  
- Meeting Scheduled  
- Dormant  
- Meeting Completed  
- No Show  
- Follow-Up  
- Decision Pending  
- Proposal / Quote Sent  
- Rep Handover  
- Not Interested  
- Post-Meeting Dormant  
- Deal Closed / Not Closed  
- Opt-out / Spam
